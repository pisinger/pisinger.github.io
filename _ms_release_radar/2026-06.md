## Security Architect Observations

- **Defense-in-depth for AI agents**: Defender XDR and Defender for Endpoint now provide local AI agent discovery and runtime protection on Windows (Preview) and macOS (Preview). Architects should plan for agent inventory visibility and runtime inspection policies as AI tool adoption grows across endpoints.
- **Identity attack surface expansion**: Defender for Identity adds 10 new security alerts spanning Entra ID, Active Directory (including DCSync), and third-party IdPs (SailPoint). The new SaaS application classifications in Security Exposure Management cover 15 critical Microsoft SaaS control planes — architects should prioritize onboarding Defender for Cloud Apps to enable these classifications.
- **Multicloud container security maturation**: Defender for Cloud extends vulnerability assessment to runtime-discovered images and node VMs on EKS/GKE, plus Kubernetes misconfiguration enforcement at admission time. Organizations running multicloud Kubernetes should reassess their container security coverage across Azure, AWS, and GCP.
- **API security posture expansion**: API security posture management now covers Function Apps and Logic Apps (GA), extending beyond API Management. Architects should inventory serverless and workflow APIs and assess exposure risks using the new recommendations.
- **Compliance and data governance**: Purview adds 9 new sensitive information types, Endpoint DLP device scoping, and external email blocking for Copilot grounding. The Insider Risk Management GA for selective AI app monitoring enables more precise DLP and IRM policy design.

---

## Security Operations Observations

- **SOC workflow changes**: Sentinel's new behavior-to-incident linking and graph exploration tools enable richer incident enrichment and relationship mapping. SOC analysts should incorporate the `BehaviorInfo` table into advanced hunting workflows.
- **New detection surface**: 10 new Defender for Identity alerts require tuning and triage playbook updates, particularly for DCSync detection, Entra ID privilege escalation patterns, and SailPoint brute-force monitoring.
- **DLP investigation improvements**: Purview's Endpoint DLP evidence collection in Data Security Investigations and Advanced Hunting access to DLP device attributes enable aggregate exfiltration analysis instead of per-alert triage.
- **Container security alert volume**: New container-level KSPM recommendations and runtime-discovered image VA on EKS/GKE will increase finding volume. SOC teams should plan for alert tuning and prioritization using the new exposure score model.
- **AI agent monitoring**: Defender's local AI agent discovery and runtime protection introduce a new alert category. SOC teams should develop runbooks for prompt injection and unsafe agent action alerts.
